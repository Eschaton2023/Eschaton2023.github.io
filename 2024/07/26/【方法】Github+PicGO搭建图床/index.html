【方法】Github+PicGO搭建图床

发表于2024-07-26|更新于2024-08-13|【方法】
|阅读量:

原理：Github可以提供接口上传保存图片，PicGo可以作为本地客户端方便的进行上传操作并提供图片URL供调用。

懒得自建可以去免费图床： 1. https://www.superbed.cn/ 2. https://imgse.com/

Github：https://github.com/
PicGO：https://picgo.github.io/PicGo-Doc/
1、先在Github上建立一个公开仓库，记下仓库名
2、在Github上右上角点头像→设置→Developer Settings→PersonalAccessToken→Tokens(Classic)→GenerateNewToken(Classic)，创建一个你需要用的期限的Token，把Token粘出来记住。
3、去PicGO下载安装客户端，图床名随便，仓库名写：Github用户名/仓库名，比如name/repository可以在Github左侧栏目看到，分支随便，Token写刚才拿到的Token，存储路径随便。保存以后图床就建好了，图片拖到上传区就可以上传，在相册里图片左下角的标志就可以复制图片URL出来。
