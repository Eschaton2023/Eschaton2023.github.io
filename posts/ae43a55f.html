【方法】AmazonAWS搭建笔记

发表于2024-11-29|更新于2024-11-29|【方法】
|阅读量:

这是大概23年的时候搞得，站已经过期了，留个笔记记录。到搭建完成后的部分更像是Docker笔记所以把后半部分放到Docker操作笔记里了。

亚马逊账户

首先需要一张VISA卡，我在申请的时候工行有免年费的星座卡，还可以挑一挑，手机上下个app就可以办，记得看一下年费信息和支持币种，免费的比较好，支持币种要包含美元。

然后去亚马逊建立账户，亚马逊因为只是免费一年所以建立账户没有什么难的，自己建个邮箱就可以建好，付款信息填刚才申请的visa卡信息。

域名

https://nic.eu.org/
去这个网站注册一个账户，然后申请新域名，在域名管理的NameService填上之后想使用的DNS解析服务器，比如如果我使用cloudflare就是填braden.ns.cloudflare.com和kristin.ns.cloudflare.com，如果是dnspod就填capella.dnspod.net和cherry.dnspod.net。具体区域不同可能使用的也不同，在申请域名前登录cloudfare或dnspod注册之后添加站点域名可以获取相关地址。

Linux服务器获取root权限

第一次还是从网页登录，之后可以putty或者xterm之类的登录
命令行界面时
su root

输入密码
使用root账户登陆图形界面
修改/etc/gdm/custom.conf文件，实现root用户免密码自动登陆
daemon下增加
AutomaticLoginEnable=True
AutomaticLogin=root

vi /etc/ssh/sshd_config
PermitRootogin yes

CentOS7 安装docker

  1. $ uname -r    //检查CentOS系统内核版本高于3.10
  2. $ sudo yum update   //确保yum包更新到最新
  3. $ sudo yum remove docker  docker-common docker-selinux docker-engine   //卸载旧版本docker
  4. $ sudo yum install -y yum-utils device-mapper-persistent-data lvm2  //安装软件包，yum-util 提供yum-config-manager功能，另外两个是devicemapper驱动依赖的
  5. $ sudo yum-config-manager –add-repo https://download.docker.com/linux/centos/docker-ce.repo    //设置yum源
  6. $ yum list docker-ce –showduplicates | sort -r  //列出仓库中docker版本
  7. $ sudo yum install docker-ce
      $ sudo yum install docker-ce-17.12.0.ce   //选自己要安装的版本
  8. $ sudo systemctl start docker
    $ sudo systemctl enable docker   //启动并加入开机启动
  9. $ docker version    //验证是否安装成功

安装yum工具

     device-mapper-persistent-data
     lvm2 –skip-broken
    
centos镜像源更改：
编辑 /etc/yum.repos.d/CentOS-Base.repo 文件
把4处网址的mirrorlist注释掉，去掉baseurl的注释，把地址替换为mirrors.tuna.tsinghua.edu.cn
dd：删除光标所在那一列

更新软件包缓存
yum makecache
yum install -y docker

配置docker镜像加速器
sudo mkdir -p /etc/docker
sudo tee /etc/docker/daemon.json <<-‘EOF’
{
  “registry-mirrors”: [“https://xfewuv42.mirror.aliyuncs.com"]
}
EOF
sudo systemctl daemon-reload
sudo systemctl restart docker
